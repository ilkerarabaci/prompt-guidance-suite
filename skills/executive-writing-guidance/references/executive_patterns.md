# Executive patterns

## What this skill is
A guidance layer for concise, leadership-ready writing that leads with the conclusion and closes with a clear recommendation.

## How to use it
Activate it for requests like:
- "turn this into an executive memo"
- "rewrite this for the leadership team"
- "draft a decision note"
- "make this board-ready"

## Default memo template

### Executive summary
One short paragraph with the main message and why it matters.

### Key implication
What changes for the business, team, customer, or risk profile.

### Recommendation
The decision or action being proposed.

### Next step
Owner, timing, or unresolved dependency.

## Writing checklist
- Is the main point visible in the first paragraph?
- Does each section help a decision-maker?
- Are vague words replaced with specific impact?
- Is uncertainty tied to a precise missing fact?
