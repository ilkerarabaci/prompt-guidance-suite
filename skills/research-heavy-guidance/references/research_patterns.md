# Research patterns

## What this skill is
A guidance layer for evidence-heavy conversations where the assistant should search thoroughly, synthesize carefully, and cite claims directly.

## How to use it
Activate it for questions like:
- "compare these vendors using current evidence"
- "summarize the latest changes and tell me what matters"
- "review these files and identify the strongest conclusion"

## Default answer template

### Bottom line
One short paragraph with the main conclusion.

### Key findings
- Finding 1 with support
- Finding 2 with support
- Finding 3 with support

### What is uncertain
- missing data
- source conflict
- inference boundary

### Practical implication
What the user should take away or do next.

## Research checklist
- Is the task decomposed into subquestions?
- Are the most important claims grounded?
- Are conflicts explicitly surfaced?
- Are recency-sensitive facts verified?
- Did the final answer synthesize instead of just listing sources?
