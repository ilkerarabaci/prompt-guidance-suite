# v1.0.0

Initial public release of the Prompt Guidance Suite.

## Included skills

- Prompt Guidance Companion
- Research Heavy Guidance
- Code Development Guidance
- Executive Writing Guidance
- Academic Writing Guidance
- Agentic Automation Guidance

## Highlights

- Added one general-purpose behavior layer for structured, reliable conversations
- Added five specialist skills for research, code work, executive writing, academic writing, and workflow-style execution
- Standardized package structure for ChatGPT skill uploads
- Added release-ready ZIP files for direct installation

## Intended use

This release is designed for users who want reusable guidance layers instead of rewriting prompting instructions for every conversation.

## Notes

- These skills are guidance layers, not guarantees
- Higher-priority platform instructions and safety controls still apply
- Future releases may refine trigger descriptions, examples, and packaging ergonomics
